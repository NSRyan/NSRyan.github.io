Cyclus

Cyclus is an agent-based nuclear fuel cycle simulator that is incredibly versatile, one of the initial core developers (Professor Katy Huff) likes to say that, "Cyclus can be used to model any process from making a grilled-cheese to international nuclear fuel cycles." The software achieves this versatility through a series of generic archetypes that are primarily transaction based. Over the years, the user community and developers have created a litany of nuclear specific archetypes for everything from proliferation assessment to fuel burnup, our contribution to the ecosystem falls under this category with EVER and CLOVER.

EVER

The Enrichment Versatile non-Equilibrium Reactor (EVER) archetype, in its lowest fidelity case, functions almost exactly like the Cycamore reactor. To compare, we have deployed EVER with a Cycamore reactor using preferences to achieve the same result and a Cycamore reactor with a single input recipe.

In this flow path, the user defines each fuel recipe based on external depletion calculations and brings that information to Cyclus.

CLOVER

The Core LOading Versatile non-Equilibrium Reactor (CLOVER) archetype, in its lowest fidelity case, introduces a new capability to Cyclus by allowing the user explicit control over assemblies inside of a particular batch of fuel entering a reactor core.
